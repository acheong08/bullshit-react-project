# Accessible Play Store - Client: Chra Abdoulqadir

## Project Overview

The aim of the project is to develop a web-based platform that allows for the discovery of accessible mobile games. 
While guidelines currently exists for making accessible games, very few platforms help disabled gamers find games that are right for them.

---

## Client Requirements

### Accessibility Needs
- Enable discovery of accessible mobile games
- Provide trustworthy accessibility information
- Support users with hearing and dexterity impairments
- Ensure intuitive, easy-to-understand design

### Requested Features
- Filter and sort by interests and accessibility features
- Voice command navigation
- Visual alternatives for all audio
- Watchlist for saving games
- Graceful, accessible error handling

---

## Our Implementation

### Accessibility Features
- **Visual Design:** WCAG AA compliant color scheme with light/dark mode toggle
- **Motor Accessibility:** Minimum 50px button sizes for dexterity-impaired users
- **Screen Reader Support:** Full compatibility across all pages with proper alt tags
- **Navigation:** All pages reachable within 3 clicks
- **Voice Control:** Complete voice command navigation
- **Game Information:** Clear, descriptive accessibility tags for each game

### Core Features
- **Search & Filter:** By genre, accessibility tags, with sortable results
- **Dual Review System:** Separate ratings for gameplay and accessibility accuracy
- **Video Content:** YouTube links default to captions enabled
- **Personalization:** User profiles save accessibility customization preferences
- **Error Handling:** Clear, accessible error messages on authentication pages
- **Bookmarking:** Save and manage favorite games
- **Reporting:** Flag incorrect game information for admin review
