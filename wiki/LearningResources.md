## Tech Stack

- **Builder**: Bun
- **Testing**: Bun test with happy-dom and @testing-library

## Programming Language: Typescript
Typescript is a super-set of Javascript, the premier language for the web. This means it offers all of Javascripts features and functions, but also it adds an additional layer on top of these: Typescript's type system.
